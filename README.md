# FitFlow

A full-stack fitness app, built one screen at a time from a complete UI design (splash → onboarding → auth → profile setup → home → workouts → trainers → appointments → premium).

## Stack
- **Frontend:** React (Vite)
- **Backend:** Node.js + Express (REST API)
- **Database:** TBD (Postgres/MongoDB)
